# Customer Churn Prediction Using Machine Learning

## Project Overview

This project uses machine learning to predict whether a customer is likely to churn. The goal is to help a business identify at-risk customers early so retention teams can take action before revenue is lost.

## Business Problem

Customer churn directly impacts revenue, customer lifetime value, and growth. By predicting churn risk, companies can prioritize retention efforts, reduce preventable customer loss, and improve decision-making.
